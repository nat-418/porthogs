Find which processes are hogging what ports!

## Usage

```
$ porthogs 8080 8000
    PID TTY          TIME CMD 
  23580 ?        00:00:23 PM2 v4.4.0: One
  13675 ?        00:00:48 PM2 v5.1.0: Another
```

## Installation

Grab a binary for your system in the `dist/` directory:

* FreeBSD x86-64
* Linux ARM
* Linux x86
* Linux x86-64
* macOS x86-64
